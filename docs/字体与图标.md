## 🔤 字体（Fonts）

Railway 官网的设计系统页面显示他们使用了一套自定义的排版系统 [[7]][[9]]：

- **标题字体**：使用 "Huge, bold, punchy" 风格的定制字体，包括 Jumbo、Large、Heading 1-6 等级别
- **正文字体**：使用系统默认的无衬线字体栈（`sans-serif`），配合 Tailwind CSS 的字体缩放系统（TextXS 到 TextXL）

⚠️ 注意：搜索结果中提到的 "Railway" 字体 [[31]][[34]][[51]] 是由 Greg Fleming 设计的开源字体（SIL Open Font License），**但这与 railway.com 网站实际使用的字体无关**，只是同名。

关于 Fonts In Use 提到的 "New Rail Alphabet (A2-TYPE, 2009)" [[32]]，那是英国铁路品牌相关的字体，**并非** railway.com 部署平台所用。

## 🚀 本项目字体配置（已优化）

### 字体方案：Inter + Inter Tight（与 Railway 一致）

| 用途 | 字体 | 说明 |
|------|------|------|
| 正文/UI | **Inter** | 可变字体，支持 100-900 字重 |
| 标题/展示 | **Inter Tight** | 更紧凑的字间距，更强的视觉冲击力 |
| 代码/终端 | **JetBrains Mono** | 可变字体，专为代码设计 |

### 使用方式

```tsx
// 正文和 UI 元素 - 默认 font-sans
<p className="text-base">正文内容</p>

// 标题 - 使用 font-heading
<h1 className="text-2xl font-bold font-heading">标题</h1>

// 代码 - 使用 font-mono
<code className="font-mono">console.log('hello')</code>
```

### Tailwind 配置

```js
// tailwind.config.js
fontFamily: {
  sans: ['Inter', 'system-ui', 'sans-serif'],      // 正文
  heading: ['Inter Tight', 'Inter', 'sans-serif'], // 标题
  mono: ['JetBrains Mono', 'monospace'],           // 代码
}
```

### CSS 全局样式

```css
/* 标题自动使用 Inter Tight */
h1, h2, h3, h4, h5, h6, .font-heading {
  font-family: 'Inter Tight', 'Inter', sans-serif;
  letter-spacing: -0.02em;
}
```
